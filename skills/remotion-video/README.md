# Remotion Video Skill

此 skill 来自 [wshuyi/remotion-video-skill](https://github.com/wshuyi/remotion-video-skill)

## 描述

使用 Remotion 框架以编程方式创建视频。Remotion 让你用 React 组件定义视频内容，支持动画、字幕、音乐可视化等。

## 触发词

- "用代码做视频"、"编程视频"、"React 视频"
- "Remotion"、"remotion"
- "/remotion-video"

## 适用场景

### 程序化视频
1. 批量生成
2. 数据驱动（如年度总结）
3. 音乐可视化
4. 自动字幕

### 教程讲解视频
5. 技术概念可视化（如 CNN、算法）
6. 分层递进讲解
7. AI 配音教程

### 3D 视频
8. 产品展示/模型动画
9. 卡通角色讲解
10. 3D 数据可视化
11. Logo 动画

## 文件说明

- `SKILL.md` - Claude Code skill 主文件
- `audioConfig.ts` - 音频配置模板
- `generate_audio_edge.py` - Edge TTS 生成脚本
- `generate_audio_minimax.py` - MiniMax TTS 生成脚本
